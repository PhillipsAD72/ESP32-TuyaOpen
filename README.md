# ESP32-TuyaOpen
Zařízení pro připojení do sítě Tuya na bázi ESP32
### Historie projektu
Poté, co jsem získal (zdarma) UUID a AUTHKEY od technické podpory Tuya, jsem zahájil svoje první funkční testy takového zařízení. Jako základ jsem využil vývojovou desku ESP32 Devkit V1, která má několik odchylek, které je třeba zohlednit ve zdrojovém kódu. Více k tomu bude uvedeno v adresáři s upravenými příklady.
